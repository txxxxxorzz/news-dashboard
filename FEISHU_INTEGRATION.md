## 🛠️ 配置步骤

### 第 1 步：创建飞书自建应用

1. **访问飞书开放平台**
   - URL: https://open.feishu.cn/
   - 登录飞书账号

2. **进入开发者后台**
   - 点击顶部导航「**企业自建**」
   - 点击「**创建企业自建应用**」

3. **填写基本信息**
   ```
   应用名称：热点情报中心
   应用图标：上传 🔥 图标（建议 512x512 PNG）
   应用描述：实时热点新闻聚合平台
   ```

4. **点击「创建」**

---

### 第 2 步：配置应用首页

1. **在左侧菜单找到「应用首页」**

2. **点击「配置」**

3. **填写配置**：
   ```
   首页类型：网页
   
   PC 端首页 URL: 
   https://your-vercel-app.vercel.app
   
   移动端首页 URL: 
   https://your-vercel-app.vercel.app
   
   页面可见范围：全员可见（或指定部门）
   ```

4. **点击「保存」**

---

### 第 3 步：配置应用菜单（可选）

1. **在左侧菜单找到「应用菜单」**

2. **点击「配置」**

3. **添加菜单项**：
   ```
   菜单名称：热点榜单
   菜单类型：网页
   页面链接：https://your-vercel-app.vercel.app
   
   菜单名称：我的收藏
   菜单类型：网页
   页面链接：https://your-vercel-app.vercel.app/?page=saved
   ```

4. **点击「保存」**

---

### 第 4 步：配置权限

1. **在左侧菜单找到「权限管理」**

2. **添加权限**：
   - ✅ 发送消息
   - ✅ 访问用户信息（可选）

3. **点击「保存」**

---

### 第 5 步：版本发布

1. **在左侧菜单找到「版本管理与发布」**

2. **点击「创建版本」**

3. **填写版本信息**：
   ```
   版本号：1.0.0
   版本描述：热点情报中心上线
   ```

4. **点击「提交审核」**
   - 自建应用通常自动通过

5. **点击「发布」**

---

### 第 6 步：添加到工作台

1. **在飞书中打开工作台**
   - 点击底部导航「**工作台**」

2. **添加应用**
   - 点击右上角「**+**」或「**添加应用**」
   - 在「自建应用」标签找到「**热点情报中心**」
   - 点击「**添加**」

3. **完成！**
   - 工作台出现 🔥 图标
   - 点击直接打开

### 配置推送通知

修改推送脚本，同时推送到飞书群和应用：

```python
# 添加到 push_feishu.py

def push_to_feishu_app(content):
    """推送到飞书应用"""
    app_id = "cli_xxxxxxxxxxxxx"  # 你的 App ID
    app_secret = "xxxxxxxxxxxxx"  # 你的 App Secret
    
    # 获取 access_token
    token_url = "https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal"
    token_resp = requests.post(token_url, json={
        "app_id": app_id,
        "app_secret": app_secret
    })
    access_token = token_resp.json().get('tenant_access_token')
    
    # 发送应用内消息
    msg_url = "https://open.feishu.cn/open-apis/im/v1/messages"
    headers = {
        "Authorization": f"Bearer {access_token}",
        "Content-Type": "application/json"
    }
    
    payload = {
        "receive_id": "all",  # 发送给所有用户
        "msg_type": "text",
        "content": {
            "text": content
        }
    }
    
    requests.post(msg_url, headers=headers, json=payload)
```

## 🆘 常见问题

**Q: 应用创建失败？**
A: 确保使用的是企业飞书账号，个人账号可能受限

**Q: 网页打不开？**
A: 检查 URL 是否正确，确保是 `https://` 开头

**Q: 移动端显示不正常？**
A: 网页已做响应式适配，检查手机浏览器兼容性

**Q: 如何分享给同事？**
A: 在应用后台设置「可见范围」为全员或指定部门
