## 🚀 第一部分：部署到 Vercel（10 分钟）

### 步骤 1：准备代码（2 分钟）

**1.1 初始化 Git 仓库**

打开终端，执行：

```bash
cd /home/admin/.openclaw/workspace/web

# 初始化 Git
git init

# 配置 Git 用户（替换为你的信息）
git config user.name "你的名字"
git config user.email "你的邮箱@example.com"

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit - 热点情报中心"
```

**1.2 创建 GitHub 仓库**

1. 打开 https://github.com
2. 点击右上角 "+" → "New repository"
3. 填写：
   - Repository name: `news-dashboard`（或其他你喜欢的名字）
   - Description: `热点情报中心 - 实时新闻聚合平台`
   - ✅ Public（公开）
   - ❌ 不要勾选 "Initialize this repository with a README"
4. 点击 "Create repository"

**1.3 推送到 GitHub**

```bash
# 关联远程仓库（替换为你的用户名）
git remote add origin https://github.com/YOUR_USERNAME/news-dashboard.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

✅ **检查点**: 刷新 GitHub 仓库页面，应该能看到所有文件

---

### 步骤 2：部署到 Vercel（3 分钟）

**2.1 登录 Vercel**

1. 打开 https://vercel.com
2. 点击 "Continue with GitHub"
3. 授权 Vercel 访问你的 GitHub

**2.2 导入项目**

1. 点击 "Add New..." → "Project"
2. 在 "Import Git Repository" 下找到 `news-dashboard`
3. 点击 "Import"

**2.3 配置部署**

保持默认设置即可：
- Framework Preset: `Other`
- Build Command: **留空**
- Output Directory: **留空**
- Install Command: **留空**

点击 "Deploy"

**2.4 等待部署完成**

- 你会看到部署进度
- 大约 1-2 分钟
- 看到绿色对勾 ✅ 和 "Congratulations" 表示成功

**2.5 获取访问地址**

部署成功后会显示：
```
https://news-dashboard-xxxx.vercel.app
```

点击 "Visit" 预览你的网站！

✅ **检查点**: 打开链接，应该能看到热点情报页面

---

### 步骤 3：配置自动更新（5 分钟）

**3.1 在服务器上配置 Git 推送权限**

```bash
# 生成 SSH 密钥（如果已有可跳过）
ssh-keygen -t ed25519 -C "your_email@example.com"

# 查看公钥
cat ~/.ssh/id_ed25519.pub

# 复制输出的内容，然后：
# 1. 打开 GitHub → Settings → SSH and GPG keys
# 2. 点击 "New SSH key"
# 3. 粘贴公钥内容
# 4. 点击 "Add SSH key"
```

**3.2 测试 Git 推送**

```bash
cd /home/admin/.openclaw/workspace/web

# 创建测试文件
echo "test" > test.txt
git add test.txt
git commit -m "Test commit"
git push origin main

# 如果成功，删除测试文件
git rm test.txt
git commit -m "Remove test file"
git push origin main
```

**3.3 配置自动部署脚本**

编辑 crontab：

```bash
crontab -e
```

修改最后一行为：

```bash
# 热点情报抓取 + 聚合 + 推送 + 自动部署 - 每 6 小时执行
0 */6 * * * cd /home/admin/.openclaw/workspace/scripts && source .env && python3 news_fetcher.py >> news_data/cron.log 2>&1 && python3 aggregate_news.py >> news_data/cron.log 2>&1 && python3 push_feishu.py >> news_data/cron.log 2>&1 && /home/admin/.openclaw/workspace/scripts/auto_deploy.sh >> news_data/cron.log 2>&1
```

**3.4 测试自动部署**

```bash
# 手动运行一次自动部署脚本
/home/admin/.openclaw/workspace/scripts/auto_deploy.sh
```

如果看到 `✅ 自动部署完成`，说明配置成功！

✅ **检查点**: 等待 1 分钟，然后刷新 Vercel 网址，应该能看到数据更新

## 📱 第二部分：飞书工作台集成（5 分钟）

### 步骤 1：创建飞书应用（3 分钟）

2. 选择 "自建应用"
3. 填写：
   - 应用名称：`热点情报中心`
   - 应用图标：上传一个 🔥 图标（或任意图片）
4. 点击 "创建"

**1.3 获取应用凭证**

创建成功后，在应用管理页面记录：
- **App ID**: `cli_xxxxxxxxxxxxx`
- **App Secret**: `xxxxxxxxxxxxxxxx`

⚠️ **重要**: 保存好 App Secret，只显示一次！

---

### 步骤 2：配置应用（2 分钟）

**2.1 添加网页**

1. 在应用管理 → 应用配置
2. 点击 "添加网页"
3. 填写：
   - 页面名称：`热点榜单`
   - 页面链接：你的 Vercel 地址（如 `https://news-dashboard-xxxx.vercel.app`）
4. 点击 "保存"

**2.2 配置权限**

1. 点击左侧 "权限管理"
2. 点击 "申请权限"
3. 搜索 `user:readonly`
4. 勾选后点击 "确定"
5. 点击 "提交审核"（通常自动通过）

**2.3 发布应用**

1. 点击左侧 "版本管理与发布"
2. 点击 "创建版本"
3. 填写版本号：`1.0.0`
4. 填写更新说明：`初始版本`
5. 点击 "提交"

## 🛠️ 常见问题解决

### Q1: Git push 失败？

**错误**: `Permission denied (publickey)`

**解决**:
```bash
# 检查 SSH 密钥是否配置
ssh -T git@github.com

# 如果失败，重新配置 SSH 密钥
ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub
# 复制输出到 GitHub → Settings → SSH and GPG keys
```

### Q2: Vercel 部署失败？

**检查**:
1. GitHub 仓库是否公开
2. Vercel 是否有仓库权限
3. 查看 Vercel 部署日志（Deployments → 点击最新部署 → View logs）

### Q3: 飞书应用打不开网页？

**可能原因**:
- Vercel 域名被墙（国内访问慢）
- 解决：使用国内 CDN 或服务器

**临时方案**:
- 在飞书里复制链接，用浏览器打开

### Q4: 数据不更新？

**检查**:
```bash
# 查看 Cron 任务
crontab -l

# 查看日志
tail -50 /home/admin/.openclaw/workspace/scripts/news_data/cron.log

# 手动测试抓取
cd /home/admin/.openclaw/workspace/scripts
source .env && python3 news_fetcher.py
```
